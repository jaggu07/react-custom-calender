## react-custom-calender

Custom calender with highlight selected date

## Demo

See the working [Demo](https://kbct5.csb.app/)

## Installation

```
yarn add react-custom-calender

```
## Usage

```
<Calender />
```

## Props

```
fromDate => date object
toDate => date object
monthsToDisplay => integer (default=3)

```
